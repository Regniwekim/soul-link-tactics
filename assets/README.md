# Assets

This directory contains visual assets for Soul-Link Tactics.

## Directory Structure

```
assets/
├── icons/          # Card artwork and system icons
├── ui/            # UI elements (buttons, panels, etc.) - TODO
├── effects/       # Visual effects (animations, particles) - TODO
└── audio/         # Sound effects and music - TODO
```

## Card Artwork Guidelines

### Dimensions
- **Card Artwork**: 512x512px (recommended)
- **System Icons**: 128x128px
- **UI Icons**: 64x64px

### Format
- PNG with transparency preferred
- SVG for icons when possible

### Naming Convention
```
card_name_variation.png
```

Examples:
- `flame_drake.png`
- `flame_drake_evolved.png`
- `aqua_warrior.png`

## System Icons

Each of the 8 systems should have a distinct icon:

- **INFERNO** (Vermilion) - Flame symbol
- **FLOW** (Dark Blue) - Wave symbol
- **AERO** (Sky Blue) - Wing/Cloud symbol
- **SPARK** (Yellow) - Lightning bolt symbol
- **HELIX** (Teal) - Leaf/DNA spiral symbol
- **TERRA** (Orange) - Mountain/Rock symbol
- **PSY** (Magenta) - Eye/Brain symbol
- **VOID** (Charcoal) - Skull/Shadow symbol

## Placeholder Assets

Until custom artwork is created, you can use:
- Solid color squares with system colors
- Simple geometric shapes
- Free game asset packs (ensure licensing allows it)

## Contributing Artwork

If you're an artist interested in contributing:

1. Follow the dimension and format guidelines
2. Ensure artwork matches the digital/holographic theme
3. Use the color palette for consistency
4. Submit via pull request with artwork in appropriate folder
