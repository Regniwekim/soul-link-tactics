## [0.1.0] - 2025-02-12

### Added
- Initial project setup with Godot 4.6
- 3x3 battle grid system for both players
- Complete card framework with all 8 systems/factions
- Base card class with all properties from GDD
- GridSlot class for grid position management
- GameBoard main controller
- Turn structure with 6 phases (Recharge, Draw, Memory, Main, Battle, End)
- Basic combat system with simultaneous damage
- Unit exhaustion and summoning sickness
- Attack type system (Melee, Ranged, Flyer, Sniper)
- Keyword framework (Blocker, Blitz, Piercing, Stealth)
- Color-blind friendly faction colors (Wong/Okabe-Ito Palette)
- Extensive code comments for beginners
- Test units for demonstration
- Complete documentation (README, Implementation Guide, GDD)

### Technical
- Modular architecture for easy extension
- Signal-based event system
- Resource-based card system
- Organized project structure
